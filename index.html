MFDC Beta Button

This is an example marketing or informational website using the MFDC beta button. It includes a large callout called a jumbotron and three supporting pieces of content. Use it as a starting point to create something more unique.

See Beta-Button docs »

Basic usage

To use the MFDC beta-button just include the code to the right somewhere in the header of your page.

<!-- MFDC Beta bar -->
<script src="http://beta.mfdc.biz/beta-button.js"></script>

Customized usage

You can override the existing button set by specifying <ul id="mfdc-beta-buttons"> somewhere in your page body.

You will also need to include the main script in the header as above.

<ul id="mfdc-beta-buttons">
	<li class="item" data-tooltip="View MFDC Basecamp page"><a href="https://mfdc.basecamphq.com"><img src="/img/basecamp.png"/></a></li>
	<li class="item" data-tooltip="Visit the MFDC.biz site"><a href="http://mfdc.biz"><img src="/img/mfdc.png"/></a></li>
	<li class="item" data-tooltip="Contact us"><a href="mailto:info@mfdc.biz"><img src="/img/email.png"/></a></li>
	<li class="item" data-tooltip="MY CUSTOM BUTTON!"><a href="http://twitter.com"><img src="/img/twitter.png"/></a></li>
</ul>
